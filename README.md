# Career-Recommendation-System
NLP-based career recommendation system using Flask and SpaCy to analyze user skills, interests, and goals and provide personalized career suggestions.

## 📌 Description

Choosing the right career can be difficult because many traditional career guidance systems rely on fixed questionnaires or keyword matching. This project solves that problem by using **SpaCy-based semantic similarity** to understand natural language input from users. The system compares user input with predefined career categories and returns personalized career recommendations through a simple web interface.

The project is developed using **Python**, **Flask**, **HTML**, **CSS**, and **SpaCy**. It follows a layered design with frontend, backend, and NLP processing components.

## ✨ Features

- Accepts user input in natural language
- Analyzes skills, interests, and goals using NLP
- Uses semantic similarity for accurate matching
- Applies rule-based filtering for better relevance
- Displays personalized career recommendations
- Simple and user-friendly web interface
- Built with open-source technologies

## 🛠️ Technologies Used

- Python  
- Flask  
- SpaCy  
- HTML  
- CSS  
- JavaScript  
- Jinja2 Templates

## ⚙️ How It Works

- User enters skills and interests
- Flask sends input to backend
- SpaCy processes and compares similarity with career data
- System recommends best matching careers
- Results are displayed on the webpage
